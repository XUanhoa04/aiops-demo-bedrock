## About

I care about systems that **fail safely** and **explain themselves** to on-call engineers.

- Focus: **AIOps**, **observability**, **SRE practices**, **AWS**, container platforms  
- Approach: closed-loop automation with **human gates** on high-risk actions  
- Currently shipping **SentinelLoop** — a production-*like* AIOps portfolio stack with topology-aware RCA and CI evaluation
